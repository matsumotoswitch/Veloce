# Veloce
## 概要 / Overview
Veloce（ヴェローチェ）は、AI生成画像に埋め込まれたメタデータ（プロンプト・生成パラメータ）を高速に解析・表示する、超軽量なデスクトップ画像ビューアー＆マネージャーです。

## 主な特徴 / Features
*  **圧倒的な起動レスポンス:** TauriによるOSネイティブWebViewの活用とRustバックエンドにより、Electron特有の重さを排除。軽量なバイナリで一瞬で起動します。
*  **AIメタデータの瞬時抽出:** PNG, WebPファイルに埋め込まれた生成プロンプトやモデル情報を抽出。Diff機能（Dキー）で2枚の画像のパラメータ比較も可能です。またMP4形式の動画にも対応し、実行ファイルと同階層に `ffmpeg.exe` を配置することで、OS非対応の動画コーデック（H.265等）であっても確実かつ瞬時にサムネイルを生成できます。
*  **高機能な独立ビューアー:** 拡大縮小、回転、左右・上下反転、ウィンドウフィット（F/Spaceキー）に対応。低解像度画像向けの独自 unsharp-filter（Uキー）を搭載。背面のDOM Swap方式により、画像送り時のチラつきや暗転ラグが排除されています。
*  **快適なシームレスファイル管理:** 仮想スクロールの実装により数万枚の巨大フォルダでも一瞬でサムネイルを表示。物理ドライブ自動検出、遅延読み込みツリー、履歴（戻る・進む）による直感的なナビゲーション。複数選択によるドラッグ＆ドロップ移動・コピー、競合チェック（上書き防止フロー）も安全に機能します。
*  **スマートフォルダとレーティング:** 画像のプロンプトやアスペクト比、レーティング（0〜5）を組み合わせた高度な検索条件を「スマートフォルダ」として保存し、SQLiteキャッシュを利用して超高速にフィルタリングできます。
*  **ブックマーク（お気に入り）機能:** よくアクセスするフォルダを色やアイコンで自由にカスタマイズし、専用のブックマークバーにピン留めすることで瞬時にアクセスできます。

## 開発環境とビルド方法 / Development & Build
### 1. 前提条件 / Prerequisites
開発およびビルドには、**Node.js** と **Rust (Cargo)** 環境があらかじめセットアップされている必要があります。

### 2. セットアップ / Setup
```bash
# リポジトリのクローン
git clone https://github.com/matsumotoswitch/Veloce.git
cd Veloce

# 依存パッケージのインストール
npm install
```

### 3.1. 開発モードでの起動 / Development
```bash
# npx 経由での起動
npx tauri dev

# または npm スクリプト経由での起動
npm run tauri dev
```

### 3.2. リリースビルドの生成 / Production Build
```bash
# npx 経由でのビルド
npx tauri build

# または npm スクリプト経由でのビルド
npm run tauri build
```
ビルドが成功すると、`src-tauri/target/release/` 配下に最適化されたスタンドアロンな実行ファイル（`veloce.exe` 等）が生成されます。
※VS Code を使用している場合、`upx.exe` を使用してビルドバイナリを自動圧縮するビルドタスク（`.vscode/tasks.json`）が用意されています。

## キーボードショートカット / Keyboard Shortcuts
メイン画面の起動中、いつでも `F1` または `H` キーを押すことで、詳細なヘルプオーバーレイを画面上に表示できます。（※独立ビューアー画面では `H` キーは画像反転アクションとなります）

### メイン画面
| キー | アクション |
| :--- | :--- |
| `F1` / `H` | ヘルプ（ショートカット一覧）の表示 / 非表示 |
| `矢印キー` | 画像の選択を移動（Shift 併用で範囲選択） |
| `Ctrl + A` | 現在のフォルダ内のすべての画像を選択 |
| `Ctrl / Shift + クリック` | 画像の複数選択 |
| `ダブルクリック` | 選択したサムネイルから独立ビューアーウィンドウを開く |
| `F5` | 最新の情報に更新（ディレクトリの再読み込み） |
| `Ctrl + F` | 検索キーワード入力欄（検索バー）にフォーカス |
| `Alt + ←` / `Alt + →` | フォルダ移動履歴の「戻る」 / 「進む」 |
| `F2` | 選択中のファイル、または左ツリーで選択中のフォルダの名前を変更 |
| `Delete` | 選択中のファイル、または左ツリーで選択中のフォルダを安全にゴミ箱へ移動 |
| `Ctrl + Z` | 直前のファイル・フォルダ操作（移動・削除など）を元に戻す |
| `Ctrl + C` | 選択中の画像をクリップボードに直接コピー（テキスト選択中はテキストコピー） |
| `A` | 開いているすべてのビューアーウィンドウを画面に横一列に整列 |
| `D` | 選択した2枚の画像のプロンプト・パラメータ情報を取得して比較 (Diffモーダル) |
| `Ctrl + Tab` / `Ctrl + PageDown` | 次のタブへ切り替え |
| `Ctrl + Shift + Tab` / `Ctrl + PageUp` | 前のタブへ切り替え |
| `0`〜`5` | 選択中の画像にレーティングを設定（0で解除） |
| `Esc` | 各種モーダル（Diff/お気に入り/スマートフォルダ編集）、ヘルプ、コンテキストメニューを閉じる |

### ビューアー画面
| キー | アクション |
| :--- | :--- |
| `←` / `→` | 前の画像 / 次の画像を表示 |
| `マウスクリック` | 左クリックで前の画像 / 右クリックで次の画像を表示 |
| `マウスホイール` | 上スクロールで前の画像 / 下スクロールで次の画像を表示 |
| `Ctrl + ホイール` | 画像のズームイン / ズームアウト（手動倍率変更） |
| `左ドラッグ` | ウィンドウの移動（非ズーム時） / スクロール（ズーム時） |
| `Ctrl + 左ドラッグ` | ズームイン時、画像内を自由に移動（パン操作） |
| `↑` / `↓` | 画像を右に90度回転 / 左に90度回転 |
| `H` | 画像を左右反転（水平反転） |
| `V` | 画像を上下反転（垂直反転） |
| `F` | ウィンドウを画像サイズに合わせる（モニターを超える場合は収まるように縮小） |
| `Space` | オリジナルの解像度（100%等倍）でウィンドウを表示 |
| `0`〜`5` | 現在表示中の画像にレーティングを設定（0で解除） |
| `F11` | ウィンドウのフルスクリーン表示切り替え |
| `A` | 開いているすべてのビューアーウィンドウを画面に横一列に整列 |
| `B` | ウィンドウ枠（アクセントボーダー）および右上コントロール群の表示 / 非表示 |
| `S` | 動画のシークバーの表示 / 非表示 |
| `U` | 画像のシャープ表示（アンシャープマスク） / 滑らか表示の切り替え |
| `Delete` | 現在の画像をゴミ箱に移動し、自動的に次の画像を読み込んで表示 |
| `Ctrl + C` | 現在表示中の画像をクリップボードにコピー（黄金のフラッシュ余韻エフェクト付き） |
| `Esc` | ビューワーウィンドウを閉じる |

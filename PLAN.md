# Veloce 現状の仕様・機能まとめ

このドキュメントは、現在の画像管理・ビューアアプリ `Veloce` が備えている機能とアーキテクチャの現状を整理するためのものです。

## 1. アプリの概要とターゲット
* **目的**: 大量の画像を快適に閲覧・整理・検索するための、超軽量かつ高パフォーマンスな画像ビューア兼管理ツール。
* **主要ターゲット**: NovelAI等のAI画像生成ユーザー。処理するメインの画像フォーマットはPNGとWebP。
* **技術スタック**: Tauri (Rust) + SQLite + Vanilla JS。仮想DOMフレームワーク（React等）は使用せず、Event Delegation等の最適化により極限の軽量化を実現。
* **動作環境要件**: Windows 8.1環境（WebView2 v109 / Chromium 109互換）での動作を保証。新しいJS配列メソッドやCSSネストなどは使用制限あり。

## 2. 主な機能一覧

### 2.1 高機能な独立ビューアー
* メインの管理画面とは完全に独立したウィンドウで画像を高速に展開。
* 拡大縮小、回転、左右・上下反転、ウィンドウフィット / 等倍（100%）表示にショートカットで対応。
* 低解像度画像向けの独自アンシャープマスク・フィルタ機能。
* 複数のビューアーウィンドウを開いている際、ショートカット(`A`キー)で横一列に整列可能。

### 2.2 ファイル管理とメタデータ表示
* 左側のツリービューによる物理ドライブの動的検出と遅延読み込み、履歴（戻る・進む）ナビゲーション。
* 複数選択からのドラッグ＆ドロップ（移動・コピー）や、競合時の上書きフロー対応。
* 右側のインスペクターペインで、選択中画像のファイルパスやプロンプト、各種生成パラメータを一覧表示。
* 複数画像のパラメータやプロンプトの差異を比較できる「Diffモーダル機能」。

### 2.3 スマートフォルダと超高速フィルタリング
* レーティング（0〜5）、プロンプト、ネガティブプロンプト、生成元（Source）、アスペクト比、画像の横幅・高さ、フォルダパスなど、多彩な条件を組み合わせ可能。
* これらを「スマートフォルダ」として保存し、バックエンドのSQLiteキャッシュを利用して数ミリ秒単位で高速に一覧を展開。
* UI上からスマートフォルダの新規作成、複製、条件の細かな編集が可能。

## 3. UI・デザインの特記事項
* Windows 8.1での文字化けを防ぐため、絵文字（⭐や✖など）は使用厳禁とし、必要に応じてSVGアイコンで代替している。
* メモリのガベージコレクション（GC）発生を最小限に抑えるため、DocumentFragmentを利用したDOMの差分更新を徹底。
